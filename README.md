✨About the Project
This project is very special to me because it's **my first project using the JavaScript Fetch API!** After 2 months of learning web development,
I built this to understand how to fetch real-world data from an external API and display it dynamically on a webpage.


How to Run
1. Clone this repository.
2. Open `index.html` in your browser.
3. Search for any image and see the results!
